## UI 设计规范

### 颜色方案
- 主色调：微信绿 #07C160
- 背景色：白色 #FFFFFF
- 次要背景：灰色 #F5F5F5
- 文字颜色：
  - 主文字：#333333
  - 次要文字：#666666
  - 提示文字：#999999

### 字体规范
- 标题：16-18sp，加粗
- 正文：14sp
- 小字：12sp

### 组件规范
- 按钮：圆角，主色绿色
- 输入框：灰色边框，圆角
- 卡片：白色背景，轻微阴影，圆角
- 悬浮球：绿色圆形，白色"快"字

## 功能需求清单

### 核心功能
1. ✅ 话术库管理（公司/小组/私人）
2. ✅ 话术搜索
3. ✅ 话术添加/编辑/删除
4. ✅ 话术分类管理
5. ✅ 悬浮窗快捷回复
6. ✅ 使用统计
7. ✅ 用户登录/注册
8. ✅ 个人中心
9. ✅ 系统设置
10. ✅ 团队管理

### 交互细节
1. 悬浮球可拖动，吸附屏幕边缘
2. 点击话术自动复制到剪贴板
3. 显示Toast提示"已复制"
4. 搜索实时过滤
5. 下拉刷新列表
