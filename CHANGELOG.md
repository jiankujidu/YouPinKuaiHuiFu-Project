# 更新日志

## [1.0.0] - 2024-05-18

### ✨ 新增功能

#### Android App
- 话术库管理（公司话术、小组话术、私人话术）
- 话术搜索功能
- 添加/编辑/删除话术
- 悬浮窗快捷回复
- 话术一键复制
- 使用次数统计
- 个人中心
- 系统设置

#### 后端服务
- 用户注册/登录
- JWT 认证
- 话术 CRUD API
- 团队管理 API
- 数据同步 API
- MySQL 数据库支持

#### PC 管理端
- 话术管理页面
- 团队管理页面
- 数据统计页面
- 系统设置页面
- 用户登录/注册

### 🛠️ 技术栈
- Android: Java + AndroidX + Room
- 后端: Node.js + Express + MySQL
- PC 端: Vue3 + Element Plus + Pinia
